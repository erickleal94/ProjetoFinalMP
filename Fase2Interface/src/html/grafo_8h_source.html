147 /***************************************************************************
148  * Função: Achar id
149  * Descrição
150  * Todo vértice armazenado no grafo possui um nome, uma string
151  * de até 100 caracteres mais o caracter zero terminal, e um
152  * identificador (id), um inteiro que é como o vértice é
153  * armazenado internamente.
154  * Essa função acha esse identificador
155  *
156  * O valor do id não deverá interessar ao usuário, porém,
157  * se algum cliente desejar criar mais utilidades para a biblioteca
158  * grafos, essa função é necessária.
159  *
160  * Vale dizer que vértices, origens, e arestas estão ordenados
161  * de menores id até maiores, logo se for sesejado que se mostrem os
162  * dados em ordem alfabética, devem-se inserir os vértices em ordem
163  * alfabética.
164  *
165  * ids são criados do menor para o maior, sendo o menor valor possível
166  * 0.
167  *
168  * Por motivos de segurança sempre se verifica se existe_vert nome antes
169  * de se usar achar_id
170  * Parâmetros
171  * meu_grafo - Deve ser passado um grafo inicializado
172  * nome - Deve ser passado uma string de até 100
173  * caracteres, mais o caracter zero terminal.
174  * Se a string tiver mais que 100 caracteres,
175  * a origem não será encontrado, uma vez que
176  * 100 é o maior número de caracteres que uma
177  * origem pode ter.
178  *
179  * Valor retornado
180  * Retorna um inteiro id. Caso não se ache o vértice nome, esse
181  * inteiro será -1, caso contrário será o valor do identificador
182  * de nome
183  * Assertiva de entrada
184  * O grafo já deve ter sido inicializado por criar_grafo(),
185  * se não for o programa pode ser interrompido.
186  * strlen(nome) <= 100
187  * Já foi checado que existe_vert(meu_grafo, nome) == TRUE
188  * Assertivas de saída
189  * Se existir um vértice nome, 0 <= id <= max(int), sendo que
190  * max(int) é o maior valor que pode ser representado num inteiro
191  * Se não existir id = -1
192  * Assertivas de saída
193  * O retorno será TRUE ou FALSE
194  * O conteúdo do grafo e da string passados não serão modificados
195 ***************************************************************************/
196 int achar_id(const grafo_priv_t *meu_grafo, int id_externo);
